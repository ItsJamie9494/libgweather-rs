#[repr(transparent)]
pub struct Info { /* private fields */ }
Expand description

GWeatherInfo provides a handy way to access weather conditions and forecasts from a Location, aggregating multiple different web services.

It includes also astronomical data such as sunrise times and moon phases.

Requests a reload of weather conditions and forecast data from enabled network services.
